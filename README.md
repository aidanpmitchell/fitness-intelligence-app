# Fitness Intelligence App

A personal fitness intelligence app to track bodyweight, calories, macros, and lifestyle data, then generate dashboards, maintenance estimates, and weekly check-ins.

## Tech Stack

- Next.js
- Expo / React Native
- Supabase
- TypeScript

## Current Progress

- [x] Monorepo setup
- [x] Web app initialized
- [x] Mobile app initialized
- [x] Supabase connected
- [x] Daily log table created
- [x] First full-stack daily log feature working
- [ ] Weight trend calculation
- [ ] Weekly check-in logic
- [ ] Apple Health integration

## Goal

Build a personal tool that combines nutrition, weight, recovery, and activity data into one dashboard to support cutting, maintenance, and performance goals.
